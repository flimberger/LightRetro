# LightRetro

A minimalistic, light colour scheme for Sublime Text with a retro inspired palette.

## Palette

beige/light: `#fff8dc` ("cornsilk")
blue:  `#0070bb` ("spanish blue")
brown: `#826644` ("raw umber")
green: `#6b8e23` ("olive drab")
red: `#bc0000`
yellow: `#ffcc00` ("amber")

background: light
foreground: blue
